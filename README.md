# ES32Lab biblioteca
Para a utilização da biblioteca ES32Lab é necessário a instalação da biblioteca responsável pelo funcionamento de display [TFT_eSPI](https://github.com/Bodmer/TFT_eSPI) desenvolvida por [Bobmer]( https://github.com/Bodmer).

**ATENÇÃO!** Logo para o funcionamento da biblioteca ES32Lab, torna-se indispensável o download da biblioteca [TFT_eSPI](https://github.com/Bodmer/TFT_eSPI).

# Endereço das GPIOs

- P_KEYBOARD = GPIO33
- P_LED_GREEN = GPIO17
- P_LED_YELLON= GPIO16
- P_LED_RED = GPIO13
- P_LED_BLUE = GPIO12
- P_POT1 = GPIO36
- P_POT2 = GPIO39
- P_VOLTAGE_SENSOR = GPIO34
- P_BUZZER = GPIO25
- P_DAC1 = GPIO25  // Se a DAC for utilizada como audío, será o canal L
- P_DAC2 = GPIO26  // Se a DAC for utilizada como audío, será o canal R
- P_I2S_DOUT = GPIO25
- P_I2S_LRC = GPIO26
- P_I2S_BCLK = GPIO27
- P_SDA = GPIO21
- P_SCL = GPIO22
- P_TX0 = GPIO1
- P_RX0 = GPIO3
- P_TX2 = GPIO17
- P_RX2 = GPIO16
- P_MOSI = GPIO23
- P_MISSO = GPIO19
- P_SCK = GPIO18
- P_SD_CS = GPIO5
